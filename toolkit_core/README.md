# toolkit_core

toolkit_core 是一个基于 HarmonyOS 的应用程序核心模块，使用 ArkTS 语言开发。

## 项目概述

该项目展示了一个基础的 HarmonyOS 应用界面，包含一个可交互的文本组件。主要功能包括：

- 显示欢迎文本消息
- 点击文本可切换显示内容
- 使用相对布局进行界面排版
- 响应式设计适配不同屏幕尺寸

## 核心功能

### 主页面 (Index.ets)

主页面包含以下特性：

1. **状态管理**
    - 使用 `@State` 装饰器管理文本状态
    - 初始显示 "Hello World"，点击后变为 "Welcome"

2. **UI 组件**
    - `RelativeContainer` 作为根容器，占满整个屏幕
    - `Text` 组件用于显示可交互文本
    - 使用资源管理系统定义字体大小

3. **交互功能**
    - 点击文本组件可更改显示内容
    - 居中对齐布局

4. **样式设置**
    - 字体加粗效果
    - 垂直和水平居中对齐
    - 使用应用资源定义字体大小

## 快速开始

1. 确保已安装 DevEco Studio
2. 克隆项目到本地
3. 使用 DevEco Studio 打开项目
4. 构建并运行项目
